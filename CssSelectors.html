CSS Selectors


01. Element selector

Element selectors, are used to target HTML elements based on their element name.

Syntax:element { /* styles */ }

Example:

                    
     HTML:
                        
     <p>This is a paragraph.</p>
     <div>This is a div.</div>
     <h1>This is a heading.</h1>
                        
                        
     CSS:
                        
     p { color: blue; }
     div { background-color: #f0f0f0; }
     h1 { font-size: 36px; }
                        
                        
     Result: The p elements will be blue, div elements will have a gray background,
             and h1 elements will be 36px.

02. Class Selector

In CSS, a class selector is a type of selector that targets HTML elements based on their class attribute.

Syntax: .className { /* styles */ }

Example:

                    
            
    HTML:
    
    <p class="highlight">This text will be highlighted.</p>
    <p>This text will not be highlighted.</p>
    <span class="highlight">This span will also be highlighted.</span>
    
    
    CSS:
    
    .highlight {
      background-color: yellow;
      color: red;
    }
    
    
    Result: Both the p and span elements with the class highlight 
            will have a yellow background and red text.

03. Id Selector

In CSS, an ID selector is a type of selector that targets a single, unique HTML element based on its id attribute.

Syntax:#idName { /* styles */ }

Example:

                    
     HTML:
     <h1 id="mainHeading">Main Heading</h1>
     <p id="intro">Introduction paragraph</p>
                        
                        
     CSS:
                        
     #mainHeading {
       color: blue;
       font-size: 36px;
     }
                        
     #intro {
       font-style: italic;
     }
                        
                        
     Result: The h1 element with ID mainHeading will be blue and 36px,
     and the p element with ID intro will be italic.

04. Universal Selector or Grobal Selector

The universal selector (*) is a CSS selector that matches all elements in the document.

Syntax:*{ /* styles */ }

Example:

                    
      HTML:
    
      <html>
       <body> 
        <h1>Hello World!</h1>
        <p>This is a paragraph.</p> 
        <div>This is a div.</div> 
       </body> 
      </html>
      
      CSS:
                        
      * { color: blue; font-size: 18px; }
                        
      Result: All elements (h1, p, div) will have blue text and 18px font size.

05. Attribute selector

Attribute selectors are a powerful way to target HTML elements based on their attributes and values.

Syntax: [attribute="value"] { /* styles */ }

Example:

                    
    HTML:
    
    <input type="text" placeholder="Enter name">
                        
                        
    CSS:
                        
    input[type="text"] { color: blue; }

06. Grouping CSS selectors(,)

Grouping CSS selectors allows you to apply the same styles to multiple elements or classes.

Syntax:selector1, selector2, ... { /* styles */ }

Example:

                    
    HTML:
    
    <p>This is a paragraph.</p>
    <div>This is a div.</div>
    <h1>This is a heading.</h1>
                        
                        
    CSS:
                        
    p, div, h1 {
      color: blue;
      font-size: 18px;
    }
                        
                        
    Result: All p, div, and h1 elements will have blue text and 
    18px font size.

07. Combinator selectors

Combinator selectors are used to target elements based on their relationships with other elements.

Types of Combinator Selectors:

  • Child Combinator (>)
  • Descendant Combinator (space)
  • Adjacent Sibling Combinator (+)
  • General Sibling Combinator (~)

I. Child Combinator

The child combinator (>) is a CSS selector that selects direct child elements of a specified parent element.

Syntax:parent > child { /* styles */}

Example:

                    
    
    HTML:
    
    <div>
      <p>First paragraph</p>
      <span>Second span</span>
      <div>
        <p>Nested paragraph</p>
      </div>
    </div>
    
    
    CSS:
    
    div > p {
      color: blue;
    }
           
    Result: Only the "First paragraph" will be blue because it is a direct child of the div.
            The "Nested paragraph" is not selected because it is a grandchild, not a direct child.

II. Descendant Combinator

The descendant combinator (a single space) is used to select elements inside elements. Selects all <p> elements inside <div> elements

Syntax:element1 descendant { /* styles */}

Example:

                    
    HTML:
    
    <div>
      <p>First paragraph</p>
      <span>
        <p>Nested paragraph</p>
      </span>
      <div>
        <p>Deeply nested paragraph</p>
      </div>
    </div>
    
    
    CSS:
    
    div p {
      color: green;
    }
                        
                        
    Result: All three paragraphs ("First paragraph", "Nested paragraph", and 
           "Deeply nested paragraph") will be green because they are all 
            descendants of the div.

III. Adjacent Sibling Combinator (+)

The next sibling combinator only selects the immedite sibling,not all subsequent siblings

Syntax:element1 + element2 { /* styles */}

Example:

                    
    HTML:
    
    <div>First div</div>
    <span>Second span</span>
    <p>Third paragraph</p>
                        
                        
    CSS:
                        
    div + span {
      color: red;
    }
                        
                        
    Result: The text "Second span" will be red because it is the immediate sibling of the first div.

IV. General sibling combinator (~)

The general sibling combinator (~) is a CSS selector that selects all sibling elements that follow a specified reference element.

Syntax:element1 ~ element2 {/* styles */}

Example:

                    
          
    HTML:
    
    <div>
      <p>First paragraph</p>
      <p>Second paragraph</p>
      <span>Span element</span>
      <p>Third paragraph</p>
    </div>
    
    
    CSS:
    
    p ~ span {
      color: red;
    }
    
    
    Result: The "Span element" will be red because 
    it is a sibling of the first p and appears after it.
